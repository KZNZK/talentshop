# 서비스 시나리오

재능 예약 시스템의 기능적, 비기능적 요구사항은 다음과 같습니다. 사용자가 원하는 재능을 예약한 후 결제를 완료합니다. 담당자가 재능 보유자를 할당합니다. 사용자는 할당 현황을 확인할 수 있습니다.
